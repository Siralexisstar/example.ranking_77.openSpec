## Project Purpose

- React UI for registering daily CrossFit WOD results.
- Daily, weekly, and monthly ranking views.
- Professional UI patterns: clear hierarchy, responsive layout, accessible forms, readable tables, and visible loading, empty, and error states.

## Future Additions

- Testing with [Vitest](https://vitest.dev/) and [Testing Library](https://testing-library.com/).
- End-to-end testing with [Playwright](https://playwright.dev/).
- Form validation library if result entry flows become more complex.
- API or serverless layer for persistent WOD results and historical rankings.
